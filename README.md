##知识储备

###编程：

熟练使用Python。Python常用的机器学习框架有Pytorch，Tensorflow， 前者入门更快且使用便捷。

最好同时掌握C++。Opencv用C++编写，对Python有较好的支持，在计算机视觉方面有非常广泛的应用。


###数学：

微积分、线性代数、概率与统计、最优化、信息论等，越偏理论的研究对数学方面要求越高
